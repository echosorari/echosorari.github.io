8.切换系统，进行语言设置，时间设置

# arch-chroot /mnt /bin/bash
# nano /etc/locale.gen

chroot（change root），就是改变程序执行时所参考的根目录的位置。同时chroot可以以限制使用权限（

bin是binary的简写，主要用于放置系统必备的执行文件；bash是Unix shell的一种，是许多Linux发行版的默认Shell（Shell俗称壳，是指为使用者提供操作界面的软件（命令解析器），他接收用户命令，然后调用相应的应用程序）。

用nano指令打开时区设置后对”en_US.UTF-8 UTF-8 ；zh_CN.UTF-8 UTF-8“进行反注释（删掉前面的#），退出保存。

# locale-gen
# echo LANG =en_US.UTF-8 > /etc/locale.conf

locale是软件正在进行时的语言环境，是根据计算机用户使用的语言、所在国家以及地域、当地文化传统所定义的一个语言环境。它包括语言、地域、以及字符集。locale-gen是编译本地定义文件的一个列表。

echo命令用于字符串的输出，可以打印普通字符串、显示转义字符、显示变量、显示换行风，此处是用来查看时间的正确性。

conf在这里是存放于etc目录下locale文件的拓展名，这个知道就行了，不用过多了了解（）。

“>”是重定向的意思，使用>会直接覆盖掉后面的文件。与之相似的>>则是再文件后方追加的意思，可以在文件中追加内容。

LANG是一种变量，用于定义Linux系统的主语系环境，默认值为安装Linux时候选择的语言。格式一般为“zh_CN.UTF-8”之类的，时区时间格式是固定的，可以自行搜索。

# ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime
# hwclock --systohc --utc //设置硬件时间

ln指令（link files）的功能是为某一个文件在另一个位置建立同步的连接。当我们需要在不同的位置存放同一个文件的时候，一般就在一个固定目录下放置该文件并用ln加上link，这样做可以节省非必要的磁盘空间。
ln -sf的含义具有指向性，比如以上就是将/etc/localtime指向/usr/share/zoneinfo……值得注意的是，我这两个目录之间有一个极其微小的空格（估计有人没看见），将前后的分开代表后方指向前方。

hwclock命令可以用来显示/设置硬件时钟命令。在Linux系统中有硬件时钟和系统时钟两种时钟。这个就是用来设置硬件时钟的。hwclock -systohc则是表示将硬件时钟调整为与目前系统时钟时间一致。

UTC是指协调世界时，又称为世界统一时间、国际协调时间。其他时间都是通过这玩意换算过来的……

9.引导系统

# pacman -S dosfstools grub efibootmgr

dosfstool（dos filesystem tool）目前从网上只能得知是一种工具， 用于ARM linux的fsck_fat磁盘检查工具代码，可以在用于修复FAT磁盘，包含FAT文件系统相关内容。应该是安装引导系统grub有关。

GRUB（GRand Unified Bootloader）是多操作系统启动程序，它允许用户可以在计算机内同时拥有多个操作系统，并且在计算机启动时选择希望运行的操作系统。这一步引导装载程序实际上是引出更高级的功能，以允许用户装载并将控制权交给特定的操作系统。

#grub-install --target=x86_efi --efi-directory=/boot/EFI --recheck

# grub-mkconfig -o /boot/grub/grub.cfg

12.安装桌面环境

# lspci | grep VGA
# pacman -S 驱动包

grep 命令用于查找文件里符合条件的字符串，这个地方是找VGA，上两行命令是安装显卡驱动。这个地方的VGA应该是指VGA接口，VGA（Video Graphics Array）叫视频图形阵列，具有分辨率高、显示速率快、颜色丰富的优点。VGA接口不但是CRT显示设备的标准接口，同样也是LcD液晶显示设备的标准接口，具有广泛的应用范围。

lspci，顾名思义，就是显示所有的pci设备信息。pci（Peripheral Component Interconnect，外设部件互连标准）是一种总线，而通过pci总线连接的设备就是pci设备了。如今，我们常用的设备很多都是采用pci总线了，如：网卡、存储等。

pacman -S是aarch Linux中常用的安装指令，不知道可以去搜索一下（我以前好像提到过这个……）

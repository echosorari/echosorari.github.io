    UnityEditor&UnityHub国际版下载使用指北

    Grayi
    2023-04-26
    Game Engine, Unity

    本文面向希望使用相对纯净环境以及不愿意接受Unity大中华区三天许可证激活的开发者。国际版Unity提供了相对更长的许可证激活周期（三个月）。需要注意的是，国际版的Unity Editor与大中华区共用进行游戏开发的时候有极低的概率出现错误。

    针对Unity大中华区对国区Unity质量堪忧的“本土化”操作，本文特提供国际版Unity的下载与使用方法以供读者使用，如有错误欢迎指出。
    附:中国Unity给开发者社区的一封信

    环境参考

    PC系统环境：Win11 21H2
    曾用测试环境：Ubuntu18.04
    代理工具：Clash for Windows v0.20.19
    Unity Hub版本：Unity Hub 3.4.2
    Unity Editor版本：Unity 2020.3.41f1

    P.S. Unity Hub对于Unix/Linux环境仅限Ubuntu16.04、18.04和CentOS 7。

    准备工作

    区分Unity国际版与Unity中国特供版区别

    为防止使用国际版Unity的开发者下载到Unity中国特供版，笔者特提供部分区分两个版本的方法以供读者区分：

    Unity国际版 Unity中国特供
    访问网址 www.unity.com www.unity.cn
    账号注册 仅需邮箱 需要邮箱，并绑定手机号
    Unity Hub版本后缀 f1 c1或c后缀存在c字母
    Unity Editor版本后缀 f1 c1或c后缀存在c字母
    许可证激活 72hrs 3months

    卸载并清除PC已经下载的Unity开发环境

    卸载已经安装的中国版本Unity Hub并且清除C:\Users\你的用户名\AppData\Roaming\目录下的Unity Hub文件夹以及Unity文件夹。

    具备VPN/科学上网的代理工具

    科学上网是所有操作的大前提，如果不具有科学上网能力，是无法注册国际版Unity ID的（无法获得相对长时间的许可证激活），后续账号登陆也无法走国际区。
    如果你使用的为VPN，可以直接进行下列步骤。
    如果你使用的为v2ray/clash等代理软件，由于如Unity Hub等软件不遵循系统代理，请按照如下方式进行操作：

    具有TUN Mode模式的代理：

    打开代理软件的TUN Mode模式。TUN 模式可以接管其流量并交由 CFW 处理，使得Unity Hub强制经过代理。

    不具有TUN Mode模式的代理：

    对于没有TUN Mode模式的代理软件，可以通过使系统全部通过浏览器HTTP/HTTPS代理的方式强制使应用经过代理。

    1. 将unity.com和unity3d.com后缀的网址加入代理列表
    2. 安装Unity Hub完毕后，建立启动脚本unity_hub_start.bat：

    注册国际Unity ID

    挂起代理后，通过https://unity.com/ 访问Unity国际官网，点击右上角图标，选择“Create a Unity ID”，输入邮箱进行注册。使用邮箱尽量为非国区邮箱，优先使用Google邮箱、Outlook邮箱进行注册。
    请注意区分注册，Unity大中华区注册需要填写并绑定手机号，Unity国际版不需要。

    下载并安装Unity Hub

    如果已经使用VPN或挂载代理，直接访问Unity官网点击下载并安装Unity Hub即可。
    下载完成后，一定记得打开代理的TUN Mode功能或者使用脚本启动！启动后首先观察左上角Unity Hub的版本号判断是否为国际版（如图），如果不是，删除C:\Users\你的用户名\AppData\Roaming\目录下的Unity Hub文件夹以及Unity文件夹并重试。

    点击登录，注意唤起界面。下图为https://id.unity.cn/ 界面：

    正确的国际版登录界面为https://id.unity.com/ 下图：

    登录过期后，如果不愿意登录也可以选择通过导入国际版激活许可证的方式使用Unity。

    下载并安装Unity Editor

    通过Unity Hub直接下载安装

    在代理模式下，理论上可以直接通过Unity Hub安装相应版本的Unity Editor，下载完毕后注意观察Editor版本后缀是否为国际版后缀。

    通过三方网站下载安装

    考虑到仍有下载到中国特供版Unity的可能，特提供一个收集官方Installer和BT种子的项目：
    Unity I Tell You
    本部分内容参考如何下载原版Unity

    使用

    通过以上步骤，你已经可以正常使用国际版的Unity Hub以及Unity Editor了。需要注意的是，在每次打开Unity Hub的时候都需要使用脚本启动或者代理工具开启TUN Mode功能，否则登录仍然有跳转国区的可能。
    希望你能够享受相对纯粹开发过程并减少外界干扰~
